# DBC Site

This repository contains the Astro frontend and Sanity CMS admin for the Dallas Bicycle Coalition website.

Template used as reference: https://github.com/sanity-io/sanity-template-astro-clean

## Tech Stack

- Astro
- Sanity CMS
- GitHub
- Cloudflare

## Development

For local setup, available commands, project structure, Sanity/GROQ data flow,
and pull request expectations, see [CONTRIBUTING.md](./CONTRIBUTING.md).

## Deployment Process

Code changes move through GitHub pull requests. Cloudflare builds preview
deployments for pull requests and production deployments after merges.

Sanity content changes can also trigger Cloudflare rebuilds so the site reflects
newly published CMS content.
